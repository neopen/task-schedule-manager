核心特性

NeoTask 提供了一套完整的企业级任务调度解决方案

高性能

基于异步IO实现，支持每秒处理数万级任务，延迟低于10毫秒

分布式

支持 Redis 作为后端存储，实现多节点分布式部署和任务调度

定时任务

支持延时执行、周期执行和 Cron 表达式，满足各种定时需求

监控告警

内置指标收集和健康检查，支持自定义告警规则

分布式锁

基于 Redis 的分布式锁，确保任务在集群中只被一个节点执行

多种存储

支持 Memory、SQLite、Redis 三种存储后端，按需选择

自动重试

任务失败自动重试，支持配置重试次数和退避策略

可观测性

事件总线 + 指标收集，支持 Prometheus 集成

简单易用

只需几行代码，即可开始使用 NeoTask

from neotask import TaskPool

# 定义任务处理函数
async def process_order(data: dict) -> dict:
    print(f"Processing order: {data['id']}")
    await asyncio.sleep(1)
    return {"status": "success", "order_id": data["id"]}

# 创建任务池
pool = TaskPool(executor=process_order)

# 提交任务
task_id = pool.submit({"id": "order_001", "amount": 100})

# 等待结果
result = pool.wait_for_result(task_id, timeout=30)
print(f"Result: {result}")
from neotask import TaskScheduler

scheduler = TaskScheduler(executor=process_order)

# 延时 60 秒执行
task_id = scheduler.submit_delayed(
    {"action": "reminder"},
    delay_seconds=60
)

# 每 5 分钟执行一次
scheduler.submit_interval(
    {"action": "heartbeat"},
    interval_seconds=300
)

# 每天 9 点执行（Cron 表达式）
scheduler.submit_cron(
    {"action": "daily_report"},
    cron_expr="0 9 * * *"
)
from neotask import TaskPool, TaskPriority, TaskPoolConfig

# 自定义配置
config = TaskPoolConfig(
    storage_type="redis",
    redis_url="redis://localhost:6379",
    worker_concurrency=20,
    max_retries=3,
    enable_metrics=True
)

pool = TaskPool(executor=process_order, config=config)

# 高优先级任务
urgent_id = pool.submit(
    {"task": "urgent"},
    priority=TaskPriority.HIGH
)

# 批量提交
tasks = [{"id": i} for i in range(100)]
task_ids = pool.submit_batch(tasks)

# 批量等待
results = pool.wait_all(task_ids)
